Alle vragen die u eventueel kan hebben op een rij!

Deze site is gemaakt voor iedereen die graag wil uitpakken met een online cv. Vandaag worden cv's vaak via mail in .pdf format verstuurd. Door gebruik te maken van deze website kan je uitpakken met je eigen online cv die je ook telkens up to date kunt houden.

Neen, YourCV zal je gegevens nooit delen met derden.

Deze website staat op een zeer bescheiden server waardoor het niet mogelijk is duizenden mensen op een kwaliteitsvolle mannier te bedienen. Door deze activatiecode is het mogelijk het aantal gebruikers onder controle te houden. Indien je echt super graag gebruik wilt maken van deze website stuur je het best een mailtje naar de webmaster.

Neen, YourCV is met liefde gemaakt en enkel de backend gebruiken is niet de bedoeling. Uiteraard houd niets je tegen de javascript te reverse engeneeren en het toch te doen.

Neen, maar toch bedankt ;-)

Het is uiteindelijk het doel om YourCv beschikbaar te stellen in 4 talen (Nederlands, Engels, Frans en Duits). Meer talen zit er jammer genoeg niet in.

De volledge source code van YourCv is beschikbaar op github, je kan daar altijd comentaar of reactie achter laten. Forken is volledig toegestaan.

Neen, er worden enkel cookies gebruikt om ervoor te zorgen dat je ingelogd blijft als je tussen paginas navigeerd. Er worden geen coockies gebruikt om je actief te tracken of reclame op maat te voorzien.
